🏗 Things I've built

Aside from my professional work, here are some side projects that I've built in my spare time.

🚗 Driving Route Finder

Tired of Google Maps tracking your location and happen to have an Arduino in your car? This driving aid calculates the optimal driving route between any two points in my home city, while implementing joystick & touch screen support.

💰 CryptoTracker

Interested to see what your 0.0001 bitcoin is worth? Checkout my cryptocurrency market tracker, which lists the top 50 coins (ordered by market cap), along with interactive trading charts to help you check the performance and execute technical analysis on the coin.

💻 Unix Shell

Do you remember your VIM keybindings? Test your skills on my unix shell clone! Just like any console/terminal found using Linux or macOS, this shell supports various custom built-in commands, along with any regular Unix shell command to interact with the operating system.

🧪 Periodic Table Database

An intuitive way to browse, search, & visualize the numerous elements on the periodic table. Built on an Arduino, it includes touch screen support, predictive search, & configurable UI layouts.

🌤 Live Weather

A web application that allows users to search for a city to query for up-to-date weather forecasts and information, built within 8 hours during a hackathon.
